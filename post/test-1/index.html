P1--跑个流程 导入素材 导出媒体

1、新建项目

  • 新建project→设置：项目名称、保存位置

  • 文件后缀.prproj

  • PR 等同于
    项目文件 超市
    时间轴 货架
    素材 商品

2、导入素材

  • Ctrl+I 文件→导入→导入素材
  • 从文件夹拖进左下角“项目”里

3、时间轴

  • 新建序列sequence→相关设置
  • 图片拖到时间轴上进行排序
  • 设置图片显示时间和音乐对应

4、播放视频

  • 右上角“节目”可以播放设置好的视频
  • 当视频里的图片太小了怎么设置？
    窗口→效果控件→双击视频中的图片可以拖动大小
  • 导出序列
    文件→导出→媒体→保存

P2--视频剪辑 入点出点 剃刀工具 帧频率 逐帧播放

1、导入视频素材

  • 视频可以直接拖进右下角“时间轴”生成序列
  • C 剃刀工具：裁剪→选择开始点→选择结束点→删除多余的片段

2、循环播放

  • 设置入点、出点、按下循环按钮
  • 右击该片段→保存为子剪辑→成为一段素材
  • 时间轴上→右击→清除入点出点→导出视频
