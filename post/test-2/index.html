面试准备（仅供参考）

  • 面试准备（仅供参考）
    • 一、自我介绍
    • 二、面试问题
      • 【1】和学校有关的问题
      • 【2】专业方面的问题
      • 【3】个人方面的问题

一、自我介绍

中英文都要准备，时间控制在3min左右

  1. 问候语
  2. 介绍自己的姓名、学校、专业
  3. 介绍自己的性格、爱好，自我认知
  4. 为什么读研、想要研究的方向、读研计划
  5. 结束语

二、面试问题

【1】和学校有关的问题

  1. 为什么想要来澳门读研？
  2. 为什么选择澳门城市大学？
  3. 你对澳门印象最深的是？
  4. 你对这个学校有什么想法和问题？

【2】专业方面的问题

  1. 你想要在研究生阶段研究的专业方向是什么？你具体的学业计划是什么样的？
  2. 你在大学里做过什么项目？你在项目里担任的角色是什么？
  3. 你在你的专业方面擅长什么？
  4. 你有什么实习经历？你从中获得了什么？

【3】个人方面的问题

  1. 在大学有什么社团经历？参加或组织过什么大型活动？
  2. 介绍一下你的家乡？
  3. 你认为学习英语很重要吗？
  4. 研究生毕业之后你有什么规划？（读博or工作）

自我介绍时也可以将下面的面试问题中的一些问题的回答放在里面。

不管你着重介绍哪个方面，记住：最重要的是突出你自己的优势（绩点/实习经历/奖项/英语水平等等）。
